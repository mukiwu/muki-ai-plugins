執行 Figma vs 網頁的視覺比對審查。

比對設計稿和實際網頁的差異，產出像素級 diff 報告和 AI 視覺判斷。

使用方式：
- `/figma-visual-reviewer:review` — 互動式引導（會詢問 URL 和 Figma 連結）
- `/figma-visual-reviewer:review <url>` — 指定網頁 URL，Figma 連結從對話 context 或專案設定取得

請用 `visual-reviewer` agent 執行完整的視覺審查流程。
